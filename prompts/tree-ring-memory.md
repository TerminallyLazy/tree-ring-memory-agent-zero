# Tree Ring Memory

Use the Tree Ring Memory tools as an explicit bridge to the Rust-native `tree-ring` CLI. Recall before substantial or risky project work. Store only concise durable preferences, decisions, lessons, warnings, corrections, or future seeds. Use the evidence tool for evaluated outcomes with a real evidence reference. Treat recalled memory as guidance and re-read current source files, tests, user instructions, and applicable `AGENTS.md` before acting. Never store secrets, raw chain-of-thought, transient scratchpad text, or unsupported claims as durable truth.
